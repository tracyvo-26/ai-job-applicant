# AI Job Applicant Tool

An AI-powered tool that helps job seekers match their resume to job descriptions and generate tailored cover letters.

Built by Tracy Vo — Master of Predictive Analytics, Curtin University.

---

## What it does

- Parses your resume and extracts key skills and experience
- Compares your profile against a job description
- Identifies skill gaps and strengths
- Generates a tailored cover letter using AI

## Tech stack

- Python
- Claude API (Anthropic)
- Streamlit
